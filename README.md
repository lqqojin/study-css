# css
* display: block;  
기본 설정 태그: <div>, <section>   
box라 생각하라  
크기 조정 가능  

* display: inline;  
기본 설정 태그: <span>, <a>, <img>
크기가 딱 사이즈 만큼  
크기 조정 불가능  

* display: inline-block;  
inline, block 2가지 속성을 가졌음  
크기 변경 가능


* display: none;  
보이지 않음, 
활용 메뉴 보이냐 안보이냐 설정에 사용

* display: flex;  
* display: grid;
