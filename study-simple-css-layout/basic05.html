                - @media screen and (min-width: 700px) { }
                - @media 쿼리 사용
                        
            * min-width: 700px
                - 700px 이유: ipad, 태블릿 viewport 700이기 때문이다.
                        
            * meta name=viewport
                - 내용이 보이는 창
                        
            * content="width=device-width, initial-scale=1.0, maximum-scale=1.0"
                - device-width: 디바이스 기본 값
                - initial-scale=1.0 기본값 100%
                - maximum-scale 확대가능 300%
